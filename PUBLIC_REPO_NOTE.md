# Public Repo Note

This folder is prepared for the public GitHub version.

- Supported public entry: `scripts/look_tongji_gui_public.py`
- Supported public Windows build: `scripts/build_windows_app.py`
- The browser-assisted self-use helper is intentionally not included here.
- Some shared GUI source still contains internal `selfuse` branches, but this public repo only supports the public build path above.
- Do not add `.env`, `state/`, `build/`, or `dist/` to GitHub.
